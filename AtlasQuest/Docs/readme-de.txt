Über AtlasQuest:
=================

AtlasQuest ist ein Addon für Atlas oder AlphaMap, das eine Liste von Quests 
mit zusätzlichen Informationen und den zugehörigen Belohnungen für jede Instanz anzeigt. 
Derzeit werden alle Instanzen und Schlachtfelder und die meisten Quests für diese unterstützt. 

AtlasQuest wurde ursprünglich von Asurn erstellt. 
Derzeit leitet Thandrenn (aka Mystery8) das Projekt.
